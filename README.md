# Cat-Rescue

Skapare:
- Sebastian - landing-page, header
- Ben - shop
- Julia - contact, footer
- Carlos - about-us

Vi har valt att skapa en plattform för kattadoption och omplacering, där vår främsta prioritet är att förmedla trygghet och omtanke för både människor och katter. På vår hemsida kan du enkelt hitta information om katter som söker nya hem, samt möjligheten att adoptera eller omplacera en katt. Vår vision är att skapa en miljö som andas förtroende, tillit och harmoni, för att säkerställa att varje katt får ett kärleksfullt och säkert hem.

Den grafiska profilen finns uppladdad i vårt repository.
